เวอร์ชั่น 1.0.1
สิ่งที่ถูกเพิ่มเข้ามาด้านซอฟแวร์
	สิ่งที่โดดเด่นในอัพเดตที่ 1.0.1
	-ระบบเว็ปผู้ใช้สามารถปรับแต่งการตั้งค่าได้ โดยสามารถเลือกได้ว่าจะให้ระบบรดน้ำให้แบบอัตโนมัติ หรือ ควบคุมด้วยตัวของผู้ใช้เอง
	-แสดงสถานะของดินและปั้มน้ำแบบ Real-Time โดยไม่ต้องคอยรีเฟสหน้าเว็ป

	การปรับปรุงคุณภาพ
	-การปรับปรุงความลดดีเลย์ของรีเลย์
	-หน้าต่างเว็ปที่ใช้งานง่าย

รายชื่ออุปกรณ์ที่ใช้งาน
	1.ESP8266WiFi ( NodeMCU 1.0(ESP-12E Module) ) x1
	2.รีเลย์ 3.3v 1ช่อง x1
	3.โมดูลปั้มน้ำDC 3-6v 3w x1
	4.เซนเซอร์วัดความชื้นในดิน แสดงสถานะ เปียก/แห้ง x1
	แปล่งจ่ายไฟภายนอก (กรณีไม่สามารถเชื่อมสายUSBได้)
	1.รางถ่าน AA 4ก้อน x1
	2.ถ่านAA x4

คำอธิบายเวอร์ชั่น
	-ในเวอร์ชั่นนี้ตัวโปรเจคยังไม่สามารถเชื่อมต่อกับโลกอินเตอร์เน็ตได้ สามารถควบคุมได้แค่อินเตอร์เน็ตใน WIFI เดียวกันผ่านเว็ปIP 192.168.50.162
